# Problem
Given two string `A` and `B`. The task is finding the length of the shortest string which has both `A` and `B` as subsequences.

## Sample
### Input
```
A = "ACDEFH"
B = "BDFGH"
```
### Output
```
8
// "ABCDEFGH" is the shortest string that has both "ACDEFH" and "BDFGH" as subsequences
```
